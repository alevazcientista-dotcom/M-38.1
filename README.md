## Sobre este módulo

Este módulo aprofunda as técnicas de **Machine Learning** voltadas para **Modelagem de Risco de Crédito**, utilizando a biblioteca **PyCaret** para autoML, pré-processamento de dados e construção de pipelines estruturados.

---

## Exercício — 38.1

**Arquivo:** `M38.1_PyCaret_AlessandraVazCardoso.ipynb`  
**Objetivos:**
- Realizar a leitura e pré-processamento de uma base de dados de concessão de crédito.
- Montar um ambiente de experimentação com `pycaret.classification`.
- Comparar a performance de múltiplos modelos preditivos (Random Forest, LightGBM, Logistic Regression, etc.).
- Otimizar, analisar (com plotagens como ROC-AUC, Feature Importance) e salvar o melhor modelo (pipeline).

### Bibliotecas utilizadas

- `pycaret` — ambiente e AutoML
- `pandas` — manipulação de dados e tabelas
- `numpy` — cálculos numéricos

1. Abra o arquivo `.ipynb` no **Google Colab** ou **Jupyter Notebook**.
